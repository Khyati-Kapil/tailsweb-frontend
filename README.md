# Assignment Workflow Portal - Frontend

React + Vite frontend for the assignment workflow portal.

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Create `.env`:
   ```bash
   cp .env.example .env
   ```
3. Run the app:
   ```bash
   npm run dev
   ```

## Notes
- Uses a single login screen for both teacher and student roles.
- Role-based dashboards are rendered after login.
- TailwindCSS is used for styling.
